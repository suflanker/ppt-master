# 国家电网标准演示模板 - 设计规格说明

## I. 模板概览

- **模板 ID**: 国家电网_标准
- **显示名称**: 国家电网标准演示模板
- **分类**: government（政府/企业）
- **适用场景**: 国家电网工作汇报、项目建设汇报、政府简报、年度总结报告
- **设计基调**: 专业、严谨、政府央企风格
- **参考来源**: ppt国网.pptx（北京洛斯达/国网汇报模板）

## II. 画布规格

- **宽高比**: 16:9
- **画布尺寸**: 1280 × 720 px
- **SVG viewBox**: `0 0 1280 720`
- **画布格式**: ppt169

## V. 页面结构（通用）

### 封面 / 结束页公共结构
- 全幅暗色背景图（`cover_bg.jpg`，深夜蓝系调色）
- 左上角品牌标志区（`brand_logo.png`），位置：(79, 58)，高度约 56px
- 左下角日期 + 汇报单位文字（白色）
- 中左侧主标题文字区（白色粗体）

### 内容页结构
- 顶部导航栏（深蓝 `#1A2B52`，高度 80px）：页面标题（白色）+ 右侧品牌徽章（`brand_emblem.png`）
- 章节识别条（叠加 `content_header_overlay.png`，高度约 60px）：章节名称（白色）
- 内容主区（白色圆角矩形，内边距 32px，起始 y≈148）
- 右下角页码

### 目录页结构
- 左侧深色边栏（宽约 380px，覆盖 `toc_bg.jpg` 且叠加 `#0F1423` 半透明遮罩）
- 左栏上部：竖排「目录」+ 横排「CONTENTS」标题
- 右侧白色内容区：带编号的章节列表（可容纳 1–8 项）
- 右侧右上角品牌徽章（`brand_emblem.png`）

### 章节页结构
- 类似封面：全幅暗色背景
- 章节编号（大字，强调色蓝 `#4874CB`）
- 章节标题（白色粗体）
- 右下装饰条（`#4874CB` 横线）

## VI. 页面类型

### 01_cover.svg — 封面
- 背景：`cover_bg.jpg` 全幅
- 标志：左上角 `brand_logo.png`
- 占位符：`{{TITLE}}`（主标题）、`{{SUBTITLE}}`（副标题）、`{{DATE}}`（日期）、`{{AUTHOR}}`（汇报单位）

### 02_toc.svg — 目录
- 左栏暗色 + `toc_bg.jpg` 底图
- 右侧章节列表
- 占位符：`{{TOC_ITEM_1_TITLE}}`…`{{TOC_ITEM_6_TITLE}}`（最多 6 章）

### 02_chapter.svg — 章节页
- 全幅暗色背景（与封面同调）
- 占位符：`{{CHAPTER_NUM}}`、`{{CHAPTER_TITLE}}`、`{{CHAPTER_DESC}}`

### 03_content.svg — 内容页
- 顶部深蓝标题栏 + 内容主区（白色）
- 占位符：`{{PAGE_TITLE}}`、`{{SECTION_NAME}}`、`{{CONTENT_AREA}}`、`{{PAGE_NUM}}`

### 04_ending.svg — 结束页
- 与封面同款暗色背景
- 占位符：`{{THANK_YOU}}`、`{{CONTACT_INFO}}`

## VII. 布局模式

| 模式 | 说明 |
|------|------|
| 文字主导 | 全文字排列，适合工作汇报、进展说明 |
| 图文并列 | 图表 + 说明文字两列布局 |
| 数据展示 | 上方大数字指标 + 下方详细说明 |
| 时间轴 | 横向时间轴 + 里程碑节点 |

| 区域 | 数值 |

## IX. SVG 技术约束

- viewBox 固定为 `0 0 1280 720`，width/height 固定为 `1280`/`720`
- 所有图片资源用相对路径引用：`./cover_bg.jpg`、`./brand_logo.png` 等
- 字体栈优先级：`Microsoft YaHei, SimHei, sans-serif`
- 不使用 `<foreignObject>`；不使用 CSS 动画
- 颜色值统一使用 HEX（不用 rgb/rgba 简写）
- 不使用 SVG filter 模糊效果
